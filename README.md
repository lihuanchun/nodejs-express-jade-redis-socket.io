
nodejs-express-jade-redis-socket.io
===================================
主要实现
1.nodejs + redis 注册 登录

2.socket.io + websocket 聊天

===================================
在工程目录下

/usr/local/bin/express

sudo /usr/local/bin/npm install -d

sudo /user/local/bin/node app.js

sudo /user/local/bin/node socket.js

浏览器
http://127.0.0.1:3000
